How the model interprets freshness

Each item has a spectral fingerprint. Degradation shifts reflectance across wavelengths, then a classifier maps these shifts to freshness categories.

  • Blue/Green bands: indicate microbial and structural changes.
  • Orange/Red bands: correlate with oxidation and breakdown effects.
  • 3-point median aggregation: dampens localized measurement noise.
